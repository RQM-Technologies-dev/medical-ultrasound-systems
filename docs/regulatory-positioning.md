# Regulatory positioning

This repository is a research software project.

- It is not a medical device.
- It is not FDA-cleared.
- It is not intended for diagnosis, treatment, or patient-care decisions.
- It is not a production clinical workflow.
- It is not clinical decision-support software.
- No output from this repository should be interpreted as medical advice or diagnosis.

Outputs from this repository are for engineering evaluation, simulation, and benchmarking only. Any future clinical use would require independent technical validation, quality-management controls, regulatory review, and appropriate clearance or authorization before deployment.
